# PPT Translator GUI

PPT Translator 的图形用户界面版本，让非技术用户也能轻松使用。

## 功能特点

- 现代化界面 - 使用 CustomTkinter 打造的深色主题界面
- 文件选择 - 支持单文件、多文件、文件夹批量选择
- 多提供商 - 支持 OpenAI、Anthropic、DeepSeek、Grok、Gemini
- 多语言 - 支持 17 种语言的翻译
- 实时进度 - 进度条、状态显示、日志记录
- 配置持久化 - 自动保存用户设置
- 安全存储 - API 密钥加密存储

## 安装要求

- Python 3.10+
- Windows 10/11

## 安装步骤

1. 下载安装包 PPT-Translator-Setup.exe
2. 双击运行安装程序
3. 按向导完成安装
4. 从开始菜单启动程序

## 使用方法

1. 选择文件 - 点击"文件选择"标签，选择要翻译的 PPT 文件
2. 配置翻译 - 点击"翻译设置"标签，选择提供商、模型、语言
3. 开始翻译 - 点击"开始"按钮，等待翻译完成
4. 查看结果 - 在"进度"标签查看进度和日志
