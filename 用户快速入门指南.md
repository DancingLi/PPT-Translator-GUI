## 🔧 第二步：安装软件

### 安装步骤：

#### 1. 找到安装包
- 打开"下载"文件夹，或桌面
- 找到 `PPT-Translator-Setup.exe` 文件

#### 2. 双击运行安装
- **双击** `PPT-Translator-Setup.exe`
- 如果弹出"Windows 已保护你的电脑"提示：
  - 点击"**更多信息**"
  - 点击"**仍要运行**"

#### 3. 跟随安装向导
- **欢迎页面**：点击"**下一步**"
- **许可协议**：勾选"**我接受许可协议**"，点击"**下一步**"
- **选择安装位置**：
  - 建议保持默认位置（`C:\Program Files\PPT Translator`）
  - 点击"**下一步**"
- **准备安装**：点击"**安装**"
- **安装中**：等待进度条完成（约 1-2 分钟）
- **完成**：
  - 勾选"**运行 PPT Translator**"
  - 点击"**完成**"

## 🚀 第三步：开始使用

### 首次启动：

#### 1. 打开软件
安装完成后，软件会自动启动。  
或者你可以在**开始菜单**中找到"**PPT Translator**"，点击打开。

#### 2. 首次设置向导（可选）
首次启动时，可能会出现设置向导：
- 选择界面语言（**简体中文**）
- 选择主题（**深色模式** 推荐）
- 配置 API 密钥（可以先跳过，稍后设置）

### 主要功能界面：

#### 1. 文件选择区
- 点击"**选择文件**"按钮，选择要翻译的 PPT 文件
- 或点击"**选择文件夹**"，批量选择多个文件

#### 2. 翻译设置区
- **选择翻译提供商**：OpenAI / DeepSeek / 智谱 AI 等
- **输入 API 密钥**：在对应提供商处获取
- **选择源语言**：中文 / 英文 / 日文等
- **选择目标语言**：英文 / 中文 / 日文等

#### 3. 开始翻译
- 点击"**开始翻译**"按钮
- 在进度区查看翻译进度
- 完成后，翻译好的文件会保存在原文件旁边

## 🔑 第四步：获取 API 密钥（重要）

### 什么是 API 密钥？
API 密钥就像是一把钥匙，让软件能够调用翻译服务。不同的翻译提供商需要不同的密钥。

### 推荐的翻译提供商：

#### 1. DeepSeek（推荐，国内可用）
**优点**：价格便宜，中文支持好  
**官网**：[https://platform.deepseek.com](https://platform.deepseek.com)

**获取 API 密钥步骤**：
1. 访问官网，点击"注册"
2. 用手机号或邮箱注册账号
3. 登录后，点击左侧"API Keys"
4. 点击"创建 API Key"
5. 输入名称（如"PPT Translator"），点击"创建"
6. **复制生成的密钥**（以 `sk-` 开头）
7. 在 PPT Translator 软件中粘贴密钥

#### 2. 智谱 AI（国内可用）
**优点**：国内服务稳定，新用户有免费额度  
**官网**：[https://open.bigmodel.cn](https://open.bigmodel.cn)

#### 3. OpenAI（国外服务）
**优点**：翻译质量高  
**缺点**：国内访问不稳定，需要特殊网络环境  
**官网**：[https://platform.openai.com](https://platform.openai.com)

### 在软件中配置 API 密钥：
1. 打开 PPT Translator 软件
2. 点击"**工具**"菜单 → "**API 密钥管理**"
3. 选择对应的翻译提供商
4. 粘贴复制的 API 密钥
5. 点击"**保存**"

## 💡 常见问题解答（FAQ）

### Q1: 安装时提示"Windows 已保护你的电脑"
**解决方法**：
- 点击"**更多信息**"
- 点击"**仍要运行**"
- 这是因为软件没有微软的数字签名，但软件是安全的

### Q2: 翻译时提示"API 密钥无效"
**解决方法**：
1. 检查 API 密钥是否复制完整（不要有多余空格）
2. 检查是否选择了正确的翻译提供商
3. 检查 API 密钥是否已过期或被删除
4. 重新在官网生成新的 API 密钥

### Q3: 翻译后的 PPT 格式乱了
**解决方法**：
- 这是正常现象，因为翻译后的文字长度可能不同
- 可以手动调整文本框大小和位置
- 或者使用 PPT 的"自动调整"功能

### Q4: 软件打开后闪退/无响应
**解决方法**：
1. 确保系统是最新版本的 Windows 10/11
2. 安装最新的 Visual C++ 运行库
3. 尝试以管理员身份运行软件
4. 检查杀毒软件是否误删了文件

### Q5: 如何批量翻译多个文件？
**解决方法**：
1. 在软件中点击"**选择文件夹**"按钮
2. 选择包含 PPT 文件的文件夹
3. 软件会自动识别文件夹中的所有 .pptx 文件
4. 点击"**开始翻译**"即可批量处理
